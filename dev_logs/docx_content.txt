These 8 upgrades take AgentCloud from 'demo-safe' to 'enterprise-procurement-approved'. 3.1 Upgrade: RS256 JWT — Asymmetric Signing SEC-01   Replace HS256 with RS256 asymmetric JWT signing What: Generate an RSA-2048 key pair. Sign JWTs with the private key (server-side only). Verify with the public key (can be shared publicly for third-party verification). Why: HS256 uses the same secret to sign and verify. If the SECRET_KEY is ever leaked (env var exposed in logs, misconfigured container), every existing token is permanently compromised. RS256 means a leaked public key is useless — only the private key signs.   → # Generate keys once   → openssl genrsa -out private.pem 2048   → openssl rsa -in private.pem -pubout -out public.pem   →    → # config.py   → JWT_PRIVATE_KEY: str  # contents of private.pem   → JWT_PUBLIC_KEY: str   # contents of public.pem   → JWT_ALGORITHM: str = 'RS256'   →    → # auth.py   → def create_token(agent_id: str, scopes: list[str]):   →     payload = {'agent': agent_id, 'scopes': scopes,   →                'exp': datetime.now(timezone.utc) + timedelta(days=7)}   →     return jwt.encode(payload, settings.JWT_PRIVATE_KEY,   →                       algorithm='RS256')   →    → def verify_token(token: str):   →     return jwt.decode(token, settings.JWT_PUBLIC_KEY,   →                       algorithms=['RS256']) Impact: Even with a leaked public key, tokens cannot be forged. Here is exactly how it works: Agent Registration:  A developer calls POST /v1/agents/register with their owner ID, the agent name, and the permission scopes they need. The server generates a unique agent_id and a signed JWT token using RS256 asymmetric encryption. Permission Scopes:  Each agent is granted only the permissions it needs. Available scopes include READ_MEMORY, WRITE_MEMORY, CALL_AGENTS, ACCESS_TOOLS, and EXECUTE_CODE. Zero-trust by default — if a scope is not granted, the action is blocked. Audit Logging:  Every action an agent takes — every API call, every memory read, every task sent — is written to an immutable audit log in PostgreSQL. Cryptographically signed, timestamped, and exportable for compliance. Token Validation:  Every inbound API request is validated against the JWT. If the token is expired, tampered with, or the action is outside the agent's scopes, the request is rejected instantly. 4.3 The Memory System — Layer 2 in Detail Agent memory is stored in three tiers, each with different retrieval strategies: Episodic memory:  Specific past interactions and events. 'Customer Acme Corp called on March 3rd and requested a refund.' Retrieved by time range and entity. Semantic memory:  General knowledge and learned facts. 'Acme Corp prefers invoices in PDF format.' Retrieved by semantic similarity using vector search. Procedural memory:  How-to knowledge. 'To process a refund, follow these 5 steps.' Retrieved when a similar task is requested. All memory is stored in PostgreSQL with the pgvector extension. When an agent writes a memory, it is simultaneously vectorized using an embedding model and stored with both the raw text and the vector representation. When retrieving, AgentCloud performs a hybrid search — combining exact keyword matching with semantic vector similarity — to surface the most relevant memories in under 50 milliseconds. 4.4 Free Tech Stack — Start at Zero Cost Service What It Does Free Limit Monthly Cost Railway Hosts your FastAPI backend 500 hours/month Rs. 0 Supabase PostgreSQL + pgvector database 500MB storage Rs. 0 Upstash Redis Agent state and pub/sub 10,000 commands/day Rs. 0 Vercel Next.js dashboard hosting 100GB bandwidth Rs. 0 Cloudflare SSL, CDN, DDoS protection Unlimited Rs. 0 GitHub Actions CI/CD pipeline 2,000 min/month Rs. 0 Groq API LLaMA AI inference for demos Unlimited dev use Rs. 0 Anthropic API Claude API for agent examples Free development tier Rs. 0 TOTAL Full production-grade infrastructure First 50 enterprises Rs. 0 4.5 Free Startup Credits You Can Claim Right Now These programs give free cloud credits to startups — no revenue required, just fill out a form: Program Credits Available How to Apply AWS Activate $5,000 to $100,000 activate.aws.amazon.com — 3 days to approve Google for Startups Up to $200,000 cloud.google.com/startup — instant approval Anthropic for Startups $25,000 API credits anthropic.com/startups GitHub Student Pack $200,000+ in tools education.github.com — needs .edu email YC Portfolio Deal (if accepted) $1,000,000+ combined Automatic for YC companies Total claimable today Up to Rs. 2.5 crore Fill 3 forms, takes 30 minutes 5.
